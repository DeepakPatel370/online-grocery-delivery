FreshMart Output Pages Worksheet

Compiled screenshots demonstrating the frontend UI layout and system outputs for the Online Grocery Delivery project.

1. Homepage Hero & Promotional Section

Description: Shows the visual entrance to the web application featuring the top announcement alert banner, glassmorphic header navigation bar (with active logo and cart badge), and dynamic interactive Bootstrap hero slides advertising weekend discounts.

2. Product Quick View Modal

Description: Illustrates the detailed view generated overlaying the products grid when a user clicks on a item. Includes large product image, ratings, dietary preferences, pricing with dynamic percentage-off badge, product description, and quantity controls.

3. Sliding Offcanvas Shopping Cart

Description: Displays the offcanvas drawer containing the product order summary. Item quantities can be adjusted inside the drawer, shipping fee parameters are dynamically updated (free over $35), and promotional codes (such as code FRESH35) can be verified and applied.

4. Checkout Transaction Order Success

Description: Visualizes the final order success modal presented after posting order items to the SQLite3 database endpoint. It provides the final order total, estimated delivery, and generates a unique order tracking code from the server.
